> ClipFlow 提供完整的 AI 视频创作工作流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐

                                                                   │
```

## 详细步骤

### 步骤 1: 上传视频

支持多种上传方式：

- 拖拽上传
- 点击选择文件
- 批量上传

| 特性 | 说明 |
|------|------|
| 支持格式 | MP4, MOV, WebM, AVI |
| 文件大小 | 最大 2GB |
| 批量数量 | 最多 10 个 |

### 步骤 2: AI 分析

系统自动分析视频内容：

| 分析类型 | 说明 |
|----------|------|
| 🌄 场景检测 | 识别视频中的不同场景 |
| 🖼️ 关键帧 | 提取具有代表性的帧 |
| 📦 物体识别 | 识别视频中的主体对象 |
| 🎤 语音转写 | 将音频内容转为文字 |

### 步骤 3: 生成脚本

根据分析结果生成脚本：

1. 选择 AI 模型
2. 选择文案模板
3. 调整生成参数
4. 生成并预览文案
5. 手动编辑优化

### 步骤 4: 视频合成

将素材智能合成：

1. 选择片段
2. 添加转场效果
3. 匹配背景音乐
4. 合成最终视频

### 步骤 5: 预览调整

预览合成结果并进行微调：

- 片段顺序调整
- 转场效果修改
- 字幕位置调整
- 音效音量调节

### 步骤 6: 导出发布

导出最终视频：

1. 选择输出格式
2. 设置分辨率和质量
3. 选择平台预设
4. 开始导出

## AI 功能模式

### 🎤 AI 第一人称解说

以第一人称视角讲述，像主播一样与观众互动。

**适合场景：**
- Vlog
- 产品评测
- 个人分享

### 📢 AI 解说

专业解说风格，适合教程、评测类内容。

**适合场景：**
- 教学视频
- 产品介绍
- 知识科普

### 🎵 AI 混剪

自动识别精彩片段，生成节奏感强的混剪。

**适合场景：**
- 精彩集锦
- 短视频二创
- 活动回顾
